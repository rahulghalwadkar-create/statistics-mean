üìö What is Mean?

The mean (or arithmetic mean) is the average of a set of numbers. It is calculated by dividing the sum of all values by the total number of values.

Basic Formula:

Mean (xÃÑ) = Œ£x / n

Where: Œ£x = Sum of all values, n = Number of values

üí° For Grouped Data, we use three methods:

  • Direct Method - Straightforward calculation
  • Assumed Mean Method - Uses an assumed value to simplify
  • Step Deviation Method - Further simplification using common divisor

üìê Method 1: Direct Method

In the Direct Method, we find the mean by multiplying each class mark (midpoint) by its frequency, summing these products, and dividing by the total frequency.

xÃÑ = Œ£(f √ó x) / Œ£f

where x = class mark (midpoint), f = frequency

üîç Which Method to Use?

Direct Method

Use when:

  • Small frequencies
  • Small class marks
  • Simple calculations needed

‚ö° Easiest to understand

Assumed Mean

Use when:

  • Large class marks
  • Want to avoid big numbers
  • Unequal class intervals

‚ö° Reduces calculation work

Step Deviation

Use when:

  • Equal class intervals
  • Large deviations
  • Want simplest calculation

‚ö° Most efficient method
